# Produto: Redimento Code Generator

## Visão Geral
O Redimento Code Generator é uma ferramenta de geração automática de código que integra com Azure DevOps para automatizar o desenvolvimento baseado em work items.

## Funcionalidades Principais
- **Integração Azure DevOps**: Recebe work items via webhook
- **Geração Automática de Branches**: Cria branches seguindo padrão `feat/{id}_{nome}`
- **Desenvolvimento Automatizado**: Utiliza IA (Gemini) para gerar código baseado nos requisitos
- **Pull Request Automático**: Cria PRs automaticamente após desenvolvimento

## Objetivo
Automatizar o ciclo de desenvolvimento desde a criação do work item até o pull request, reduzindo tempo manual e aumentando produtividade da equipe.

## Público-Alvo
Equipes de desenvolvimento que utilizam Azure DevOps e desejam automatizar processos de desenvolvimento baseados em work items.
